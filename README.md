# Market18



## QuickStart

### init
```bash
#fill config
$ cp .env_EXAMPLE .env 
$ cp database/config.example.json database/config.json

$ npm i

# database migrate
$ npx sequelize db:migrate
```



### Development

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### Deploy

```bash
$ npm start
$ npm stop
```
